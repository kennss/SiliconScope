### The Fleet menu-bar icon is gone.

- 4.0.0 added an always-on Fleet item to the menu bar with no way to turn it off, while every other menu-bar item (CPU, GPU, Memory, Network, SSD, Sensors, Battery) is opt-in and off by default. It's removed. ([#31](https://github.com/kennss/SiliconScope/issues/31))
- The fleet is unchanged and still one click away in the window's **Devices** sidebar.
