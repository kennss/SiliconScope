### Machines with no GPU now actually appear.

- 4.0.2 fixed the decoding, but the overview tile still required a GPU — so a Raspberry Pi (or CPU-only server, or VM) connected and reported, yet stayed on "Connecting…". ([#33](https://github.com/kennss/SiliconScope/issues/33))
- Tiles now draw **CPU/RAM**, which every machine has, and show GPU/VRAM only when there's a GPU. The Linux detail view drops the GPU chart entirely rather than showing an empty one.
