### SiliconScope now watches your other machines too.

- **Fleet monitoring.** Run a small agent on a headless Mac, a Linux GPU box, or a cloud instance, and it joins the same dashboard — discovered automatically on your LAN, no IP configuration.
- **A remote Mac gets the full dashboard** — E/P cores, GPU, **Neural Engine**, Media, memory bandwidth, power, fans. No other tool shows a remote Mac's ANE.
- **A Linux/NVIDIA box gets a GPU-centric view** — utilization, VRAM, power against the card's limit, which processes hold VRAM, and loaded Ollama models.
- **Fleet overview** puts every machine on one screen, This Mac first, with paired GPU+VRAM and CPU+RAM graphs.
- **Encrypted and paired.** TLS with a bearer token, and the agent's certificate is pinned on first connect.
- **One line to install, one paste to pair** — the installer prints a single `sscope://pair…` link; paste it into "Add machine…". The Mac agent needs no sudo.

Nothing about using SiliconScope on a single Mac changes — the dashboard is the same one, with a collapsible Devices sidebar added. Collapse it and you're back to 3.x exactly.
