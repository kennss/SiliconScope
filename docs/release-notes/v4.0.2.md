### Remote machines with no GPU now work.

- The Linux agent sent `"gpus": null` on any box without `nvidia-smi` — a Raspberry Pi, a CPU-only server, a VM — and the viewer rejected the whole payload, so the machine never showed up. ([#33](https://github.com/kennss/SiliconScope/issues/33))
- Fixed on both sides: the agent always sends an array, and the viewer now treats a null or missing array as empty. **Machines running the older agent recover on their own** — you don't need to reinstall it.
